# React CCDC-App
A prototype view for crystal structure information which has been published and 
registered with DataCite by the CCDC.

## Usage

### Clone Repository

```
git clone https://github.com/TomCS3/CCDC-App
```

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### To build for production

```
npm run build
```
